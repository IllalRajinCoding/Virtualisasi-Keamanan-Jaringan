## Environment Configuration

Proyek ini dibuat untuk kebutuhan praktikum/laboratorium Virtualisasi dan Keamanan Jaringan.

Seluruh konfigurasi `.env` disertakan di repository agar memudahkan proses deployment dan demonstrasi. Kredensial yang digunakan hanya berlaku pada lingkungan VirtualBox internal dan **bukan** merupakan kredensial produksi.
